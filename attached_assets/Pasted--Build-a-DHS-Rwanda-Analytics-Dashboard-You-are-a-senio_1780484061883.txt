# Build a DHS Rwanda Analytics Dashboard

You are a senior full-stack engineer, data visualization specialist, and UX designer.

We are building a production-ready analytics dashboard for the DHS Rwanda API. The API already exists and exposes demographic, health, education, fertility, mortality, nutrition, and socioeconomic indicators.

Your job is to build a modern, highly interactive dashboard that allows policymakers, researchers, NGOs, government analysts, and development partners to explore indicators visually.

## Technology Stack

Frontend:

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS
* shadcn/ui
* TanStack Query (React Query)
* Recharts
* Leaflet or Mapbox for maps
* Framer Motion for animations

Backend:

* Existing DHS API (already built)
* Consume API endpoints directly

## Primary Goal

Create a dashboard that transforms raw DHS indicators into meaningful insights through interactive visualizations.

The dashboard should feel similar to:

* World Bank Data Explorer
* UNICEF Data Portal
* Rwanda Open Data Portal
* WHO Observatory Dashboard
* DataReportal Analytics
